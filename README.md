# Maver : Math solver 
A web based whiteboard calculator ,that allows to draw and solve mathematical equations.

# Demo 
Check the uploaded demo.mp4 to see the demo

# How to run 
1. Open two terminals
2. run `chmod +x run_model` and similaryly for run_ui script
3. run `./run_ui script`
4. modify the run_model script accordingly for installation of packages
5. finally run `./run_model`
6. Open the site on `3000` port on localhost

If you face any issue, feel free to contact me
